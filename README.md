## Botium Toys Security Audit

## Objective

Conducted a security audit to analayze, identify risks and vulnerabilities, compliance gaps (SOC type 1 and SOC type 2) and assess compliance with relevant standards and regulations for a fictional company Botium Toys. The goal is to provide actionable recommendation that enhance the company's security posture while demonstrating practical skills in risk assessment, threat analysis, and professional documentation, preparing for real-world challenges.


## Skills Demonstrated

- Risk assessment
- Security auditing
- Threat analysis
- Documentation

## Tools Used

- Google Sheets
- Markdown

## Highlights

- Identified gaps in access control, employee training and encryption noted compliance with EU regulations
